Pines Form Examples

Welcome to the Pines Form example page. This page functions as a reference for Pines' XHTML form design. It also makes a great sample design to use in your own web project. All the code on this page is licensed under the GNU Affero GPL, as it is part of the Pines project.

Pines Form was designed with the goal of providing beauty, flexibility, customizeable design, hack/JavaScript free code, and sensible, valid XHTML1.0/XHTML1.1/CSS. You can use the buttons at the bottom of this page to validate it as XHTML 1.0 Strict, XHTML 1.1, and CSS 2.1.
