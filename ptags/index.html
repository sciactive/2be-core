Pines Tags jQuery Plugin

Pines Tags is a JavaScript content tag editor plugin for the jQuery JavaScript library, developed by Hunter Perrin as part of Pines. It is licensed under the GNU Affero GPL. It is designed to provide maximum flexibility, while still being easy to use and easy to implement. Try out different themes using the selector in the top right corner of this page.

Features

  • Preserves original input element. The value is updated when tags change, so you don't have to alter your server side script.
  • Tags can be edited by clicking them.
  • XHTML 1.0 Strict compliance.

Installation

Installation is simple. Include jQuery, jQuery UI (only the CSS is required), and the CSS for a jQuery UI theme, then the following files:

  • jquery.ptags.js
  • jquery.ptags.default.css
