Pines Grid jQuery Plugin

The Pines Grid plugin was developed by Hunter Perrin as part of Pines. There are other grid plugins for jQuery, but none that satisfy all the requirements of Pines. Pines Grid's features include:

  • Pagination.
  • Filtering.
  • Sorting.
  • Customizeable toolbar.
  • Column resizing.
  • Column hiding.
  • Hierarchical data.
  • Selecting.
  • Saving and loading state. (Current page, filter, etc.)
  • Adding and removing items dynamically.
  • Selecting and deselecting items dynamically.
  • XHTML 1.0 Strict compliance.
  • Clean, sensible, easy markup.
  • Population either statically or dynamically.
  • Integration with jQuery UI.

Try out different themes using the selector in the top right corner of this page.
